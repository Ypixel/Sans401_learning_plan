## 计划

参考资料共 35个 Module ，预计划每天学习一个Module，根据具体每日学习部分难易，时间多少，适当调整学习进度，预计一个月左右学习完成。计划修改，对于一些简单的课程，略读即可，提高学习效率。保证一个月内可以完成整个课程的学习。
